# Ast Tuition Manager

## Project Documentation

Ast Tuition Manager is a comprehensive platform designed for managing tuition classes, including student enrollment, scheduling, and progress tracking. 

### Features
- **Student Enrollment:** Allow students to enroll in various classes.
- **Class Scheduling:** Schedule classes based on instructor availability.
- **Progress Tracking:** Track student performance and progress over time.
- **User Management:** Admin and user roles for better management.
- **Notifications:** Notify students and parents about class schedules and updates.

## Setup Instructions

### Prerequisites
- Node.js (version x.x or higher)
- MongoDB (version x.x or higher)
- Git (version x.x or higher)

### Installation
1. **Clone the Repository**  
   `git clone https://github.com/zahidniloy25-hue/ast-tuition-manager.git`

2. **Navigate to the Project Directory**  
   `cd ast-tuition-manager`

3. **Install Dependencies**  
   `npm install`

### Configuration
1. Create a `.env` file at the root of the project.
2. Add your MongoDB URI and other configurations as per the sample `.env.example` file.

### Running the Project
- To start the application, run  
  `npm start`

### API Documentation
For detailed API documentation, refer to the `docs` directory.
